# 구현 기능 목록
- 사용자에게 자동차 이름 입력받기
- 시도할 횟수 입력받기
- 각 차수별 실행 결과 출력
- 최종 우승자 출력

# 구현 순서
1. 사용자에게 입력을 요구하는 문장 출력
    - 잘못된 값 입력시 IllegalArgumentException 발생
2. 시도할 횟수 입력
    - 잘못된 값 입력시 IllegalArgumentException 발생
3. 각 자동차에 대해 0에서 9 사이의 무작위 값을 구해 값이 4 이상일 경우 전진
4. 최종적으로 전진한 거리가 가장 긴 자동차를 출력, 우숭자가 여러 명일 시 쉼표를 이용해 구분
